## 上午 - AI 日报修正

### 问题与修复
用户反馈 AI 日报格式问题，多次修正：

1. **链接格式错误**: 使用 `[原文]()` 隐藏链接 → 修正为直接显示完整 URL
2. **URL 不对**: 使用 placeholder 域名 → 修正为真实链接
3. **论文摘要缺失**: 补上作者、arXiv ID、贡献描述（参考 2-19 格式）
4. **标题链接不对应**: 重新用 Tavily API 获取真实数据

### 最终格式
- 热点新闻: 5条，带访问状态标记 (✅/⚠️/❌)
- GitHub Trending: 0个（不凑数原则）
- 论文: 7篇，含完整摘要
- 访问状态帮助用户判断可访问性

## 关键决策

1. **不凑数原则**: AI 新闻和 Trending 不为填满数量而记录低质量内容

4. **待分类机制**: 新增仓库不自动分类，避免误分类，需人工确认
